Import Courses from Teachable

Import Email List

Supported platforms: Mailchimp (Audience → Export → All Contacts as CSV), ConvertKit, Substack, or any CSV with an email column.

Expected columns: email, first_name, last_name (optional), status (optional — unsubscribed contacts are skipped automatically).
